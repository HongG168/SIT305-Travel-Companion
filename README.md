## Project Description
This Android app is a Travel Companion App developed for SIT305 Pass Task 2.1.  
It helps international travellers convert important values.

## Features
- Currency conversion
- Fuel conversion
- Temperature conversion

## Technologies Used
- Android Studio
- Kotlin
- XML Layout

## Conversion Categories
1. Currency: USD, AUD, EUR, JPY, GBP
2. Fuel: MPG, km/L, Gallon (US), Liter, Nautical Mile, Kilometer
3. Temperature: Celsius, Fahrenheit, Kelvin

## How to Run
1. Open the project in Android Studio
2. Run the app on an emulator or Android device
